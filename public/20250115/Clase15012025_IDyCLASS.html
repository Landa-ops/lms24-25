Definición de ID

En HTML, tenemos un atributo que se llama "id" y que se referenica como Identificador.
En un html se puede utilizar un id para identificar un único elemento o etiqueta
dentro de un documento con el fin de aplicarle una serie de características concretas.
Es decir, no debería de haber otro elemento con el mismo nombre de identificador (id) dentro
del documento HTML.

Cuando vayamos a utilizar un id en HTML, utilizaremos la siguiente estructura:
nombre_etiqueta (por ejemplo, img) id = "nombre_identificador"

Ejemplo:

img id = "foto" src=...

Definición de Class

El valor del atributo class, a diferencia del valor del atributo id, puede ser utilizado en más de un elemento de nuestro documento HTML.
Esto es muy beneficioso cuando tengamos que aplicar los mismos estilos a diferentes elementos, dado que nos permitirá reducir las líneas de código en nuestro archivo CSS.

Cuando vayamos a aplicar una clase en HTML, seguiremos la siguiente estructura:
nombre_etiqueta class = "nombre_clase":

Por ejemplo:

img class="foto" src=...
